## Install

```sh
$ deno install -f https://raw.githubusercontent.com/bernardini687/looker/master/src/looker.ts
```

## Usage

```sh
$ looker '{"header":"function","name":"info","description":"Display info.","parameter  1":null}'
#====  FUNCTION  =======================================================================
#          NAME:  info
#   DESCRIPTION:  Display info.
#  PARAMETER  1:  ---
#=======================================================================================
```
